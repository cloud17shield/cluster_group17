Project Description

This project will use the data from a competition on Kaggle website to train an algorithm to predict how quickly is a pet adopted, according to the text information (age, breed, color, maturity size, and so on) and/or the image information (photos of corresponding pets). The image part of the training set is 1.7GB; the image part of the test set is 400MB. Therefore the total data used in our 8-VM cluster will be larger than 2GB. The prediting function will be implemented in a web UI, users can upload a picture of a pet and/or some text information of this pet. the predicting results of how many days before this pet can be adopted will be returned on web UI after users click “predict” button.
